# Meta分析图表生成器

一键生成符合顶级期刊要求的Meta分析图表

## 核心能力

- 森林图 (Forest Plot)
- 漏斗图 (Funnel Plot)
- 亚组分析图
- 敏感性分析图
- GRADE证据总结表

## 快速开始

    git clone https://github.com/MoKangMedical/meta-analysis-figures.git
    cd meta-analysis-figures
    pip install -r requirements.txt
    python src/plot.py --data results.csv --type forest
